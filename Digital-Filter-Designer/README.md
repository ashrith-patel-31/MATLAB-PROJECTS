# Digital Filter Designer using MATLAB

## Overview

Digital Filter Designer is a MATLAB-based Digital Signal Processing (DSP) project developed to demonstrate noise reduction and signal recovery using filtering techniques.

The project generates a signal, adds random noise, applies a low-pass filter, and analyzes the signal behavior in both time and frequency domains using FFT.

## Features

* Generation of clean signals
* Addition of random noise
* Noise reduction using low-pass filtering
* Signal recovery after filtering
* Time-domain signal analysis
* Frequency-domain analysis using FFT
* Comparison between noisy and filtered signals

## Concepts Used

* MATLAB Programming
* Digital Signal Processing (DSP)
* Signal Generation
* Noise Modeling
* Digital Filtering
* Fast Fourier Transform (FFT)
* Time Domain Analysis
* Frequency Domain Analysis

## Methodology

The project follows these steps:

1. Generate a sinusoidal signal
2. Add random noise to simulate signal disturbance
3. Apply a low-pass filter to reduce noise
4. Compare original, noisy, and filtered signals
5. Perform FFT analysis to observe frequency components
