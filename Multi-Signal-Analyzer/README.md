# Multi-Signal Analyzer using MATLAB

## Overview

Multi-Signal Analyzer is a MATLAB-based signal processing project developed to generate, analyze, and process different types of signals.

The project demonstrates basic Digital Signal Processing (DSP) concepts including signal generation, noise addition, FFT analysis, and filtering.

## Features

* Generation of different signals:

  * Sine Wave
  * Square Wave
  * Triangle Wave
  * Sawtooth Wave

* Noise simulation using random noise generation

* Frequency analysis using Fast Fourier Transform (FFT)

* Noise reduction using filtering techniques

* Visualization of signals in time domain and frequency domain

## Concepts Used

* MATLAB Programming
* Signal Generation
* Time Domain Analysis
* Frequency Domain Analysis
* Fast Fourier Transform (FFT)
* Digital Filtering
* Noise Modeling

## Tools Used

* MATLAB R2026a
* Signal Processing Toolbox
