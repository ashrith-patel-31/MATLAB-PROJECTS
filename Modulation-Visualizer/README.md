# Modulation Visualizer using MATLAB

## Overview

Modulation Visualizer is a MATLAB-based communication system simulation project that demonstrates the generation and analysis of different modulation techniques.

The project explains how information signals are transmitted using carrier signals in communication systems.

## Features

* Generation of message signal
* Generation of carrier signal
* Amplitude Modulation (AM)
* Frequency Modulation (FM)
* Noise addition to modulated signals
* Signal visualization in time domain

## Modulation Techniques Implemented

### Amplitude Modulation (AM)

In AM, the amplitude of the carrier signal is varied according to the message signal.

### Frequency Modulation (FM)

In FM, the frequency of the carrier signal changes according to the message signal.
