# MATLAB-PROJECTS

A collection of MATLAB projects developed for learning and applying concepts in Signal Processing, Digital Signal Processing (DSP), Communication Systems, and other Electronics & Communication Engineering (ECE) applications.

## Objectives

* Strengthen MATLAB programming skills
* Understand signal processing concepts through practical implementation
* Apply communication system fundamentals using simulations
* Build a professional engineering portfolio
* Showcase ECE-related projects on GitHub

## Projects Included

### 1. Multi-Signal Analyzer

Generation and analysis of different signal types such as:

* Sine Wave
* Square Wave
* Triangle Wave
* Sawtooth Wave

Features:

* Signal generation
* Noise addition
* Time-domain analysis
* Frequency-domain analysis using FFT
* Signal filtering

### 2. Modulation Visualizer

Simulation of communication system modulation techniques.

Topics Covered:

* Amplitude Modulation (AM)
* Frequency Modulation (FM)
* Signal visualization
* Carrier and message signal analysis

### 3. Digital Filter Designer

Design and implementation of digital filters.

Features:

* Low-pass filtering
* High-pass filtering
* Noise reduction
* Signal recovery

## Tools Used

* MATLAB
* Signal Processing Toolbox
* Communications Toolbox (where applicable)

## Learning Outcomes

Through these projects, the following concepts are explored:

* MATLAB Programming
* Signal Generation
* Signal Analysis
* FFT (Fast Fourier Transform)
* Noise Modeling
* Digital Filtering
* Communication Systems
* Data Visualization
